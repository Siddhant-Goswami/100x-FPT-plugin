# Deck Conventions (house style)

This file is loaded at the start of every deck session and grown automatically by the
Stop hook whenever you correct deck work. Edit freely; the hook only appends.

## Voice
- No em dashes. Ever. Use colons, commas, semicolons, or parentheses.
- Plain, direct, spoken. Short sentences. One idea per line of narration.
- Avoid "genuinely", "honestly", "actually".
- Confidence without hype. Name the boring true answer and defend it.

## Pedagogy
- Why before what. Derive before name. Problem before solution.
- Earn each concept with a question whose obvious answer is wrong.
- Resolve every claim to a mechanism the audience can point at, not a vibe.
- Feynman test: a labeled box is not an explanation; the annotation next to it is.

## Structure
- Cold open, the promise, the first question, then at most three principles, then the close.
- Each principle ends with: Run-tonight, Takeaway (boxed), Source (boxed), Bridge.
- The close maps each principle back to the failure the audience wrote down.

## Diagrams
- Use Excalidraw for every bracketed visual cue.
- Color is meaning: blue = input, red = core problem, green = solution, yellow = takeaway.
- The annotation carries the insight, not the box label.

## Citations
- Verify every source against the live web the day of the talk.
- Quotes under 15 words, one per source, paraphrase by default.
- Primary sources over aggregators.
